Student Project: Downhill Bike

Here is the project I conducted regarding 3d gameplay using Unity during my 2nd Year of University.

Project Details

Platform

  • Windows

Engine/Language

  • Unity(C#)

Project

  • Individual project
  • 3d, bike-riding game

About Project

This game was developed as part of my 2nd year Games Technique Course. For this assignment we were tasked with creating a 3d model version of the 2d racing game known as Stickman Downhill, which involves players trying to navigate the stickman on a bike to reach the finish line.

For this game, I decided to go for a full, 3d rendering of the game, allowing for more direction and control for the player. I added a traditional timer and life counter to the game's canvas, but only had time to supplement the timer, as it was a core gameplay mecahnic in the 2d version of the game. Additionally, to keep it consistant with the 2d origin, I designed and moulded the enviroement to involve bumps, turns and other obstacles for more diffculty. I also added a checkpoint system which would move the player's spawning position upon reaching it.

I only managed to make a woking main menu, moveable player and the game timer, with the respawning, check points, finish line and life counter having a multidue of bugs that need fixing. While this was due to numerous factors, I feel that had I more time and assitance, I would have sorted out these issues to create a fully functional game.

Student Project: Crash Drive 2

Here is the project I conducted regarding multiplayer interfaces using Unity during my 3rd Year of University.

Project Details

Platform

  • Windows

Engine/Language

  • Unity(C#)

Project

  • Individual project
  • 3d, multiplayer, vehcile stunt game

About Project

This game was developed during my 3rd year at Games Technique Course. For this assignment we were tasked with creating a 3d, multiplayer car-stunt game based off another multiplayer stunt game known as Crash Drive 2, allowing multiple players to select their own vehcile and perform various tasks for points.

For this game, I first developed the two levels, one based on a desert landscape, the other an artic region, before then setting up the network connection need for the game to connect to the server that would house the game and the mutliple players playing on it. Keeping in line with the actual game this project is based off of, I created a garage section to allow players to select their vehicles for the level, as well as a small section that can allow players to join already running servers of the game or host a game of their own.

Though I was unable to incorporate any actual gameplay or the end screen, as well as accumulate a number of errors and bugs, I feel that I have succeeded in my original objective to create two levels that allow for multiplayer functionality, such as allowing multiple players/clients to connect to the server, allowing clients to connect to the project from different machines or networks if wanted, as well as setup the possibility to run the game on a multitude of servers for better gameplay.
